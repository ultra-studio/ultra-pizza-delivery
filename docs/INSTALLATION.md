# Installation Guide

## Resource Setup

1. Place the resource in your FiveM `resources` directory.
2. Ensure required dependencies are started before this resource.
3. Review `config/shared.lua` and `config/server.lua`.
4. Add `ensure ultra_pizzajob` to `server.cfg`.
5. Restart your server and test the delivery workflow.
